Tech Support Tools
This is a tarball that uncompresses to a directory containing a bunch of usefull shell scripts for tech support purposes. The main ones of interest are ts_network_backup which automates the network backup process and ts_identify_backups which is used in conjunction with it (given a ticket number it will identify the directory it was backed up to.
The latest version can always be found at:
http://tsbackup/tstools.tar.gz.
The current version is tstools.0.1.6.tar.gz.
Eventually it will be come a deb that be installed via apt-get form the freegeek archives. N.B. This is still alpha quality, bugs may be present though it is definitely usable. Bug reports are welcome via email to paulm@freegeek.org
Music...
Access Ampache -- a web based music player and play music in Tech Support (assuming tsbackup is plugged into the speakers).
Flash fix
A small shell script that “fixes” issues with Adobe Flash and AMD CPU's. This isn't really a fix, more of an ugly workaround and will often break on updates. The issue is an upstream bug introduced by Adobe who apparently don't know how to use a compiler properly and so no sign of fixing it any time soon.
